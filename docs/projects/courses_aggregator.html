Aggregator of courses

March 2025

A course aggregator of modern professions. Every page touching a profession contains a set of courses of different private schools including info about the course itself (price, description, rating, etc). The page will be also providing with a list of advantages and some data from HeadHunter on wages ( a russian analog of Indeed ).
